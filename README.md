# Terraria-Chest Tweaks

仿写 Terraria 的箱子整理机制，实现的一个 Mod
##功能：
1. 打开一个箱子，玩家背包中带着物品 A，箱子中也存放着物品 A 时，可以快速堆叠
玩家身上所有的物品 A 至箱子中。
2. 同样的，也可以快速将箱子中所有的物品 A 取出到玩家背包中。
3. 可以在打开背包 GUI 时，将背包中的物品快速堆叠至附近的箱子中，当然，前提是箱子中必须有相同的物品了。简单说，就是高级版的第一个功能。
